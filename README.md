# 🌌 Star Wars Unity Game

A 2D sci-fi game inspired by the Star Wars universe, built using Unity.

## 🎮 About the Game

This project is a space-themed game featuring a futuristic environment, smooth gameplay mechanics, and immersive visuals inspired by iconic Star Wars interiors.

Currently in early development — core mechanics and environment design are being built.

## 🛠️ Built With

* Unity (Game Engine)
* C#
* Aseprite (for pixel art assets)

## 🚧 Current Progress

* ✅ Basic scene setup
* ✅ Space-themed background environment
* ⏳ Player movement (in progress)
* ⏳ Combat system (planned)

## 🎯 Future Plans

* Lightsaber combat / blaster mechanics
* Enemy AI (stormtrooper-style)
* Level design with multiple maps
* Sound effects & music
* UI & health system

## 📂 Project Structure

* `Assets/` → Game assets, scripts, scenes
* `ProjectSettings/` → Unity settings

## 🚀 How to Run

1. Clone the repository
2. Open the project in Unity Hub
3. Open the main scene
4. Press Play ▶️

## 📌 Notes

This is a learning + portfolio project, focused on improving game development skills and building a polished Unity game.
